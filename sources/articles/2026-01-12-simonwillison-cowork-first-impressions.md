# First impressions of Claude Cowork, Anthropic's general agent

Simon Willison's review of Claude Cowork published January 12, 2026 — the day of Cowork's launch.

## Key observations

**Overall assessment**: "a general agent that looks well positioned to bring the wildly powerful capabilities of Claude Code to a wider audience."

**Technical characterisation**: "Claude Cowork is regular Claude Code wrapped in a less intimidating default interface and with a filesystem sandbox configured for you without you needing to know what a 'filesystem sandbox' is." Verdict: "a really smart product."

**Availability at launch**: Research preview for Max subscribers ($100 or $200/month plans), via the Claude Desktop macOS app. Anthropic's description: "Claude Code for the rest of your work." Available to $20/month Pro subscribers from January 16, 2026.

**Practical demo**: Willison had Claude search his own site (simonwillison.net), running 44 individual searches to determine which of his drafts had already been published.

**Security concern — prompt injection**: "the problem with prompt injection remains that until there's a high profile incident it's really hard to get people to take it seriously." Willison noted he himself has "all sorts of Claude Code usage that could cause havoc if a malicious injection got in."
