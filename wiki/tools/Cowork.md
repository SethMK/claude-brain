# Cowork

## What it is

Anthropic's product that productises the **AI coworker** framing for non-developers. Mac desktop app with sandboxed long-running agent sessions. ([[Mike Krieger]]'s primary example of [[Agent-native product|agent-native product design]].)

Available to Max subscribers ($100/$200/month) at launch (Jan 12, 2026), later extended to Pro ($20/month, Jan 16, 2026). Described by Anthropic as "Claude Code for the rest of your work."

## Simon Willison's take (Jan 2026)

[[Simon Willison]] reviewed Cowork on launch day ([[2026-01-12-simonwillison-cowork-first-impressions]]):

- **What it is technically**: "regular Claude Code wrapped in a less intimidating default interface and with a filesystem sandbox configured for you without you needing to know what a 'filesystem sandbox' is" — verdict: "a really smart product."
- **Strategic framing**: "well positioned to bring the wildly powerful capabilities of Claude Code to a wider audience."
- **Unresolved risk**: prompt injection. "The problem with prompt injection remains that until there's a high profile incident it's really hard to get people to take it seriously." Willison has "all sorts of Claude Code usage that could cause havoc if a malicious injection got in."
